# Hello :wave:
Welcome to **Chit Chat App** Project. It is a E-grocery shop.<br/>
An android application sharing posts and instant messaging.<br />
## Technology/ Stack used :mortar_board:
- Java 
- XML
- firebase
- Android Studio
